ABOUT THIS PROJECT

The story behind this Swiss Punk Typography educational website.

The Project

This website was created as part of the IS 117 - Introduction to Web Development course at the New Jersey Institute of Technology (NJIT) during the Fall 2025 semester.

The assignment challenged students to create an educational website exploring a design movement or style of their choice. I chose Swiss Punk Typography because of its rebellious spirit and lasting influence on modern design—from album covers to websites.

Design Decisions

The visual design of this site attempts to embody Swiss Punk principles while remaining accessible and readable. Key decisions include:

Color Palette: The bold red, yellow, and blue against black backgrounds reference the high-contrast, energetic aesthetic of Swiss Punk posters.

Typography: Oswald (a condensed sans-serif) for headlines creates the bold, impactful type Swiss Punk is known for. Space Mono for body text adds the technical, grid-breaking feel.

Layout: While the site uses modern CSS Grid, elements like skewed text, asymmetric compositions, and bold borders nod to the movement's rule-breaking nature.
